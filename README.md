## About

Simple PHP tool to quickly check for sane webhosting setup.

This is mainly about testing if (effective) `server process owner == file owner`.

## Usage

1. Upload `hostinfo.php` to your webhosting server
2. Navigate to the file in your web browser (e.g., `http://yourdomain.com/hostinfo.php`)
3. Review the displayed information about file permissions and ownership
4. Use this information to verify proper webhosting configuration
